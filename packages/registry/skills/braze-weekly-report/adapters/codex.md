# Codex Adapter — Braze Weekly Performance Report

Use the `braze_list_campaigns`, `braze_get_campaign`, `braze_list_canvases`, `braze_get_canvas`, and `braze_list_segments` MCP tools to:

1. Confirm workspace scope and date range
2. Pull top 5 campaigns by volume
3. Flag anomalies (CTR drop >20%, failure rate >2x)
4. Generate wins/risks/actions summary

Output should be a Slack-ready message with bullet points.
